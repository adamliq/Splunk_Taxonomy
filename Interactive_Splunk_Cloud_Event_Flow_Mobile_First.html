UF queue and forwarding path

Winparsing and structured parsing feed the parsingQueue, then the event moves to the tcpout queue.

winparsing structuredparsing parsingQueue tcpout_queue
outputs.conf example
[tcpout]
defaultGroup = splunkcloud

[tcpout:splunkcloud]
server = inputs.example.splunkcloud.com:9997
useACK = true
sslVerifyServerCert = true
↓ TLS + ACK

Splunk Cloud Ingest Processor

The managed service receives the stream, terminates TLS and applies pipeline rules.

Receive Filter noise Mask PII Route
What the pipeline can do
• Drop low-value or repetitive events
• Mask usernames, email addresses, tokens or IPs
• Route matching events to the Splunk Cloud indexer tier
• Route selected events to Amazon S3
